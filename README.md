### AUTOHIDROPONIC
** Anderson Danilo **

El proyecto esta inclinado al cultivo hidproponico en lugares pequeños y 
y aislado como lugares residenciales, el objetivo principal es la producción 
y desarrollo de la planta de marihuana, enfocada para la satisfaccion de
consumidores eventuales o recetados.

** Tecnologias **
Se implementara el prototipo sobre una sistema embebido, bajo un procesador
Omega-onion y un SO OpenWRT Linux, y un microcontrolador (esclavo
del procesador) STM32 arquitectura ARM; el micro estara configurado y programado
bajo ChibiOS RT. 

** Licencias ** 
Todo se realizara bajo software Libre, accesible para todos.
